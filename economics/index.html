01 · The model

Borrow rate in, lender yield out

Two forces set the band. A floor: the rate must be high enough that the USDC yield beats Treasuries by a premium, or no one supplies the vault. A ceiling: the rate must stay below CDI so the loop keeps positive carry, and far enough below that the borrower's risk-adjusted return stays attractive. Everything recomputes from the inputs.

02 · The tradeoff

Lender yield rises, borrower Sharpe falls

The borrow rate is the dial. Push it up and the lender earns more, but the borrower's risk-adjusted carry thins, and past a point borrowers unwind and utilisation drops, taking the lender yield with it. The viable window is shaded.

05 · Methodology

The model, fully derived

FX as the only risk, and how leverage amplifies it

efixDI holds a fixed 1-BRL price, so a position's USD value moves only with BRL/USD. Model the pair as geometric Brownian motion: over horizon T the log-return is X ~ Normal(μT, σ²T), with σ the annual implied vol. A position at leverage L has collateral L·E in BRL and debt (L−1)·E in USD, so equity carries an FX exposure levered by L. A BRL move x changes equity by L·x.

Borrower return and the borrow-rate ceiling

L = 1 / (1 − LTV0)
Expected annual return on equity:   E[R](r) = L·CDI − (L−1)·r + L·μ
Annual return volatility:   σR = L·σ  ·  Sharpe(r) = E[R](r) / σR
Loss probability over horizon T:   P(loss) = Φ( −Sharpe(r) · √T )

The loop accrues carry linearly in T while FX risk grows as √T, so the carry dominates over the holding horizon. Setting a target loss probability p over the holding horizon gives the comfort rate, with z = Φ⁻¹(1 − p):

rcomfort = [ L·(CDI + μ) − z · L · σ / √T ] / (L − 1)

The floor and the lender yield

Lender yield(r) = r · U · (1 − f)   [ U = utilisation, f = performance fee ]
Floor yield = UST + premium  →  rfloor = (UST + premium) / ( U · (1 − f) )
Carry ceiling:   r < CDI   (positive carry per loop)

The viable borrow band runs from rfloor to the CDI ceiling, mapped through the yield equation for the lender band. Where the comfort rate sits below the floor, the operating point accepts a borrower loss probability above the set tolerance, the honest cost of a vol regime this size at this leverage. Lower leverage moves the comfort line to the right.

Liquidation probability over a window

Liquidation buffer (from the Stress Matrix):   dliq = 1 − LTV0 / LLTV
P(liquidation over T) = Φ( ( ln(1 − dliq) − μT ) / ( σ √T ) )

Carry-crash skew (the vanguard correction)

Carry trades are short FX volatility, and BRL/USD shows negative skew and excess kurtosis, so the Gaussian understates the far left tail. The Cornish-Fisher expansion adjusts the quantile used in the tail:

zCF = z + (z² − 1)·S/6 + (z³ − 3z)·K/24 − (2z³ − 5z)·S²/36

With negative S and positive excess K, the deep left-tail quantile grows more extreme than Gaussian. In practice the same effect is captured by stressing the implied vol upward by roughly 1.3 to 2 times for the operating ceiling, the conservative reading worth presenting to a risk team. Φ and Φ⁻¹ here are an erf approximation and the Acklam rational approximation, computed in-page.

Who bears what

The FX loss probability on this page is the borrower's, taken on purpose for the carry. The USDC lender does not carry it. Lender loss is the bounded liquidation tail in the Stress Matrix: it appears only if BRL gaps past the liquidation buffer faster than the position clears, which needs a move far beyond any weekly or monthly figure here.
