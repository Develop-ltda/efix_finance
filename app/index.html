◆Visão Geral do Produto

Plataforma DeFi que permite brasileiros acessarem rendimentos otimizados do CDI com alavancagem segura via protocolos de lending.

1
Entrada via PIX ou BRTH
Usuário deposita BRL via PIX (instantâneo) ou BRTH (stablecoin). PIX vai direto para conta da SPE no HausBank.
↓
2
Compra de Cotas do Fundo DI
SPE Efix automaticamente compra cotas do Fundo DI via API HausBank. Rendimento: CDI ~ 15% a.a.
↓
3
Mint do Token efixDI
Smart contract minta tokens efixDI 1:1 com BRL depositado. Token é yield-bearing (acumula rendimento automaticamente).
↓
4
Alavancagem com USDC/USDT
Usuário pode usar efixDI como colateral para pegar empréstimo em USDC/USDT (via Aave/Compound). Taxa: ~5% a.a.
↓
5
Loop de Yield
USDC emprestado é convertido em BRL e reinvestido no Fundo DI, gerando mais efixDI. Spread: 15% - 5% = 10% por loop.
↓
6
Resgate a Qualquer Momento
Fundo DI D+0 permite resgate instantâneo. Queima efixDI → Resgata cotas → PIX para usuário.

%Cálculo do APY com Alavancagem

Fórmula Base:

APY_total = CDI_base + Σ(spread × LTV^n) para n loops

Parâmetros:
• CDI_base = 15% a.a. (atual)
• LTV = 0.75 (75% loan-to-value)
• Custo empréstimo USDC/USDT = ~5% a.a.
• Exposição cambial mitigada via hedge

Exemplo com 3 loops:
Loop 0: 15% sobre 100% = 15.00%
Loop 1: (15% - 5%) sobre 75% = 7.50%
Loop 2: (15% - 5%) sobre 56.25% = 5.63%
Loop 3: (15% - 5%) sobre 42.19% = 4.22%
Total: ~32.35% APY (antes de fees e hedge)

✓ Vantagem USDC: Taxa baixa = Maior spread
⚠ Considerar: Custo de hedge cambial ~2-3% a.a.

◈Papel do BRTH no Ecossistema

O BRTH funciona como gateway de entrada para holders do token, mas a alavancagem vem de lending pools globais de USDC/USDT por questões de eficiência.

✓ BRTH para Capital

  • Entrada simplificada para holders
  • Liquidez instantânea BRL ↔ BRTH
  • Integração com ecossistema Efix
  • Utility adicional para o token

$ USDC para Alavancagem

  • Taxa de 5% a.a. (vs 8-10% BRTH)
  • Liquidez global massiva
  • Integração com Aave/Compound
  • Spread de 10% maximiza APY
Comparação de Cenários:

Alavancagem com BRTH (8% custo):
• Spread: 15% - 8% = 7%
• APY final: ~27%

Alavancagem com USDC (5% custo + 2% hedge):
• Spread líquido: 15% - 5% - 2% = 8%
• APY final: ~30%

Conclusão: USDC oferece +3% APY adicional mesmo com hedge
